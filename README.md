# 樓層圖轉 3D 列印模型

一個基於瀏覽器的 2D 樓層圖繪製 + 3D 預覽 + STL 匯出工具，專為 3D 列印設計。

## 功能特色

- **2D 繪圖**：繪製牆壁、階梯、房間、傢俱、尺寸標註
- **門窗系統**：點兩點裁切牆壁，自動產生門洞／窗洞
- **3D 預覽**：即時 Three.js 3D 預覽，可旋轉／平移／縮放
- **STL 匯出**：匯出實心牆體 STL 模型，適合 3D 列印
- **背景圖**：上傳樓層平面圖作為描圖底稿，可調整透明度／比例／固定
- **圖層系統**：依物件類型分層管理，可切換可見度
- **存檔／載入**：儲存專案為 JSON 檔（含背景圖），支援重新載入編輯
- **工作區域**：250×250mm 標準列印範圍

## 快速開始

1. 開啟 `floorplan-to-3d.html`（建議用 Chrome 或 Edge）
2. （選用）📷 上傳樓層平面圖作為描圖底稿
3. 選取工具 → 在畫布上繪製
4. 點「3D 預覽」查看立體模型
5. 調整列印設定 → 「匯出 STL」

## 工具說明

| 工具 | 快捷鍵 | 說明 |
|------|--------|------|
| 選取 | `V` / `S` | 選取／拖曳物件 |
| 牆壁 | `W` | 點擊拖曳繪製牆壁（自動約束水平／垂直） |
| 門 | `D` | 在牆上點兩點裁切出門洞 |
| 窗 | `Shift+W` | 在牆上點兩點裁切出窗洞 |
| 階梯 | `Shift+S` | 點擊拖曳繪製階梯區域 |
| 房間 | `R` | 點擊拖曳繪製房間區域（半透明標示） |
| 傢俱 | `F` | 點擊拖曳放置傢俱（可選床／沙發／桌子） |
| 尺寸 | `M` | 點擊拖曳標註尺寸（含箭頭） |
| 刪除 | `E` / `Delete` | 點擊物件刪除 |
| 平移 | 空白鍵（按住） | 拖曳平移畫布 |

> 選取物件後按 `Delete` 或 `Backspace` 也可刪除。

## 門窗系統

### 門（裁切牆壁）

1. 選取「門」工具
2. 在牆壁上點擊**第一個點**（會出現黃色標記）
3. 在牆壁上點擊**第二個點**（兩點之間的牆段被移除）
4. 牆壁會被分割為兩段，中間形成門洞

> 門洞只是牆壁缺口，不產生門框或門板模型。

### 窗（牆上開窗）

1. 選取「窗」工具
2. 在牆壁上點擊**第一個點**
3. 在牆壁上點擊**第二個點**（兩點之間形成窗洞）
4. 窗戶參數可在側邊欄調整：
   - **離地高度**：窗戶下緣離地板高度（預設 10mm）
   - **窗高**：窗戶開口高度（預設 20mm）
5. 2D 畫面上窗戶會以**藍色線段**標示
6. 3D 預覽和 STL 匯出會自動產生窗洞**鏤空**

> 提示：點擊時會自動**吸附到最近的牆壁**，即使滑鼠稍微偏離也能精準定位。若兩點落在不同牆壁會自動重來。

### 共同行為

- 門和窗都使用**兩點裁切**機制
- 兩點需落在**同一面牆**上
- 裁切寬度小於 5 單位會被忽略
- 修改窗戶參數會**即時更新**所有已存在的窗戶

## 滑鼠操作

| 操作 | 說明 |
|------|------|
| 左鍵拖曳 | 繪製物件／選取拖曳 |
| 滾輪 | 縮放畫布（以滑鼠位置為中心） |
| 空白鍵＋拖曳 | 平移畫布 |
| 點擊選取 | 選取物件（可拖曳移動） |

## 3D 預覽

- 點擊上方「3D 預覽」切換
- 滑鼠左鍵拖曳：旋轉視角
- 滑鼠右鍵拖曳：平移視角
- 滾輪：縮放
- 畫面色調：深色背景，上方 directional light 搭配陰影
- 支援物件：
  - **牆壁**：含窗洞鏤空（下方牆段＋上方牆段）
  - **階梯**：階梯狀堆疊
  - **房間**：半透明地板標示
  - **傢俱**：床（矮）、沙發（中高）、桌子（圓柱）、其他（方塊）
  - **底板**：可開關

## STL 匯出

1. 點擊「匯出 STL」
2. 設定：
   - **檔名**：匯出檔案名稱
   - **單位**：mm / cm / m
   - **品質**：高／中／低（目前無差異）
3. 點擊「匯出」下載 `.stl` 檔案
4. 牆壁為**實心**（無中空、無填充），直接可用於 3D 列印
5. 可選擇包含**底板**（厚度可調）
6. 窗戶開口會正確反映在 STL 模型中
7. Z 軸朝上（符合 3D 列印機慣例）

## 側邊欄設定

### 牆壁屬性

| 參數 | 預設 | 範圍 | 說明 |
|------|------|------|------|
| 厚 | 3mm | 1–50mm | 牆壁厚度 |
| 高 | 100mm | 1–500mm | 牆壁高度 |

> 修改牆壁屬性會**套用到所有牆壁**。

### 窗戶屬性

| 參數 | 預設 | 範圍 | 說明 |
|------|------|------|------|
| 離地高 | 10mm | 0–200mm | 窗戶下緣離地高度 |
| 窗高 | 20mm | 5–300mm | 窗戶開口高度 |

> 修改後**即時更新**所有已存在的窗戶。

### 列印選項

| 參數 | 預設 | 說明 |
|------|------|------|
| 底板 | 開啟 | 在模型底部加入平板（使用背景圖尺寸） |
| 板厚 | 2mm | 底板厚度 |
| 縮放 | 1x | 整體模型縮放比例 |

## 背景圖功能

1. 點擊「📷 上傳樓層圖」選取圖片
2. 圖片會自動置中並縮放至工作區域
3. **拖曳**圖片移動位置
4. **拖曳角落綠色方塊**調整大小
5. 調整**不透明度**（0–100%）以便對照描繪
6. 調整**比例**（1px = N mm）校準尺寸
7. 點擊「📌 固定」鎖定位置（固定後無法拖曳）

> 固定前的圖片有綠色虛線邊框和角落控制點；固定後變為藍色虛線邊框。

## 圖層系統

| 圖層 | 顏色 | 說明 |
|------|------|------|
| 牆壁 | 紅色 `#e94560` | 牆壁物件 |
| 階梯 | 紫色 `#9b59b6` | 階梯物件 |
| 房間 | 綠色 `#2ecc71` | 房間區域標示 |
| 傢俱 | 灰色 `#95a5a6` | 傢俱物件 |

- 點擊 👁 / 🙈 切換**圖層可見度**
- 只會繪製可見圖層的物件

## 專案存檔

### 儲存

- 點擊「儲存」按鈕
- 匯出為 `.json` 檔（檔名：`floorplan_YYYY-MM-DD.json`）
- 包含：所有物件、背景圖（base64）、視圖設定、參數設定

### 載入

- 點擊「載入」按鈕 → 選取先前儲存的 `.json` 檔
- 自動還原：物件、背景圖、視圖位置、所有設定
- 背景圖會非同步載入，載入完成後自動重繪

## 開發

### 技術棧

- 純 HTML + CSS + JavaScript（單一檔案，無建置工具、無框架）
- Three.js `r0.164`（CDN 載入，用於 3D 預覽與 STL 匯出）
- Canvas 2D API（2D 繪圖）
- 無外部依賴（除 Three.js CDN 外）

- **v0.1**：初始版本，含牆壁、門窗裁切、3D 預覽、STL 匯出、背景圖、圖層

### 座標系統

- 2D 畫布：世界座標單位為 mm，經由 `translate(offsetX, offsetY)` 和 `scale(zoom)` 轉換
- 3D 場景：Three.js Y 軸朝上，牆壁旋轉使用 `rotation.y`
- STL 匯出：Z 軸朝上（標準 3D 列印座標），牆壁旋轉使用 `rotation.z`
- 所有尺寸在匯出時乘以 `S.printScale` 和單位轉換因子

## 注意事項

- 建議使用 Chrome 或 Edge 瀏覽器
- 3D 預覽需要 WebGL 支援
- STL 匯出檔案大小取決於模型複雜度
- 牆壁厚度建議不小於 1mm，否則 3D 列印可能失敗
- 座標單位為 mm，工作區域 250×250mm 對應多數 3D 列印平台尺寸
- 背景圖以 base64 嵌入 JSON，大圖片可能導致存檔肥大
